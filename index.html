Security & Privacy

        
Why you can trust ZeroOffice

      


      

        
🔒 Zero Data Collection

        

          
Your documents?
            
Stay with YOU. Never uploaded. All processing happens in your browser.

          
Your files?
            
Stay with YOU. PDF, images, everything — processed locally, never sent anywhere.

          
Your API key?
            
Stored ONLY in your browser's localStorage. We never see it, never touch it.

          
Your data?
            
We don't want it. Even for free.

        

      


      

        
🔓 Offline Tools — Completely Private

        

          
✓ PDF Merger, Splitter, Image tools — zero network requests

          
✓ No internet required for offline tools

          
✓ Everything runs in your browser — only you see it

        

      


      

        
✨ AI Tools — Transparent

        

          
ℹ️ AI tools send your text/images directly from your browser to Anthropic API — we are not in the middle

          
ℹ️ Your API key goes directly to Anthropic — ZeroOffice never receives or stores it on any server

          
✓ Governed by Anthropic's Privacy Policy

        

      


      

        
❌ What We DON'T Do

        

          
❌ Don't collect IP addresses

          
❌ Don't log your operations or files

          
❌ Don't send your files to any server

          
❌ Don't sell your information

          
❌ Don't set tracking cookies

          
❌ Don't show advertisements

          
❌ Don't use hidden scripts

          
⚠ We DO use Google Analytics for anonymous page view counts only — no personal data

        

      


      

        
💾 Local Storage Only

        

          
Your preferences (theme, language, API key) are stored ONLY on your device

          
Nothing is transmitted to our servers

          
Clear everything anytime: browser Settings → Clear browsing data → Site data

        

      


      

        
📖 100% Open Source

        

          
All code is visible on GitHub

          
Anyone can audit it for security

          
No hidden trackers. No malicious scripts.

        

      


      

        
🔍 Verify Yourself

        

          
1. Press F12 → Open DevTools

          
2. Go to Network tab

          
3. Use any offline tool (PDF Merger, Image Compressor, etc)

          
4. Look at Network requests

          
✓ You'll see ZERO requests to external servers (offline tools only)
